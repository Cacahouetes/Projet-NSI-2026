# Guide d'utilisation

## Démarrage

Double-cliquez sur `main.py` ou lancez :
```bash
python main.py
```

Le launcher s'ouvre avec 3 options.

---

## Card Simulator

### Ouvrir un coffre
1. Cliquez **"Ouvrir un coffre"**
2. Choisissez une catégorie (MEME, MOTS, OBJETS, PERSONNAGES, CONCEPTS) ou **OMNI** (toutes catégories, plus cher)
3. Les 10 cartes se révèlent une par une, cliquez pour tout révéler d'un coup
4. Vendez les communes directement depuis l'écran de résultat

### Gérer son inventaire
- **Clic** sur une carte pour la sélectionner (border bleue)
- **"Vendre sélection"** pour vendre les cartes choisies
- Boutons rapides : **"Vendre Communes"**, **"Vendre Rares"**, **"Vendre Épiques"**
- Filtres par rareté et catégorie en haut

### Fusion
1. Sélectionnez 3 cartes de même rareté dans la grille (colonne droite)
2. Les cartes apparaissent dans les 3 slots (colonne gauche)
3. Vérifiez le taux de réussite et le coût
4. Cliquez **"Fusionner"**

### Shop
- 3 cartes disponibles, renouvelées toutes les 3 heures
- Cliquez sur une carte pour confirmer l'achat
- Timer affiché en bas de l'écran

### Récompense quotidienne
- 1 récompense par jour
- Le streak augmente si vous vous connectez chaque jour
- Après 48h sans connexion, le streak repart à 0

## Arcade Game

| Touche | Action |
|--------|--------|
| ← → | Se déplacer |
| ↑ | Sauter |
| Clic gauche | Tirer |
| Molette | Changer d'arme |
| R | Rejouer après la mort |
| Échap | Retour au launcher |

**Collectibles :**
- Point jaune -> +10 score + 1 pièce en DB
- Santé -> +20% de vie
- FlyGun -> nouvelle arme (tir rapide + recul)
- AK-47 -> nouvelle arme (rafale automatique)

Les ennemis deviennent plus nombreux et plus forts à chaque vague. Les ennemis forts (plus grands) infligent plus de dégâts.

## WebStats

Cliquez **"Ouvrir"** dans le launcher - une page s'ouvre dans votre navigateur.

- **Accueil** : présentation du projet et des probabilités
- **CardDex** : toutes les 756 cartes, recherche et filtres
- **Statistiques** : sélectionnez un joueur pour voir ses stats détaillées

---

## Reset de sauvegarde

Dans le launcher, cliquez **"Effacer la sauvegarde"** (bouton rouge en bas à gauche).

- Un résumé de la progression actuelle s'affiche
- Saisissez le nom du nouveau joueur
